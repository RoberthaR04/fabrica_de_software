**Objetivos do Sistema**
O objetivo geral do sistema é incentivar a prática do descarte correto do lixo, tornando assim os ambientes públicos mais limpos. 
Ações por meio da prefeitura em beneficiar os moradores e comércios que dispõem de locais adequados para descarte do lixo, seja por meio da diminuição da taxa de lixo ou IPTU.

**Conclusões finais**
O desenvolvimento do sistema Meu Ambiente permitiu aplicar na prática conhecimentos de desenvolvimento web, modelagem e organização de sistemas. 
A plataforma foi criada para informar os usuários sobre tipos de resíduos, apresentar indicadores ambientais e permitir o registro de denúncias. 
Dessa forma, o projeto contribui para a conscientização ambiental e para o aprendizado prático de conceitos importantes da Engenharia de Software.
